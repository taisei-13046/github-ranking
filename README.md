### アプリの内容▼

### サイトURL
https://github-rankiing.firebaseapp.com/

#### タイトル: Github Ranking
#### 概要: 努力を可視化するアプリケーション
     Githubのcommit履歴をもとに週間ランキング化。友人同士で競い合う
     
### アプリ開発の経緯（開発理由)▼

サポーターズハッカソンvol.11において、「エンジニアの技術力を可視化する」というテーマをもとに作成した

### 使用した技術
- React JS
- Firebase(Firestore)
- Github API
